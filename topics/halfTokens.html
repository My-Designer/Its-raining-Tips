More tokens cause more rain, up to a limit defined by the previous setting. This setting determines how fast higher tips approach that limit.

The previous setting defined the largest possible rainfall, that is, the upper limit for the size of any rainfall. Half of that limit is called a medium sized rainfall.

This setting defines how many tokens are needed to get a medium sized rainfall. Less tokens cause less rain, just like a visitor would expect. More tokens cause slightly more rain, getting nearer to the limit.

Больше токенов вызывает больше дождя, до предела, определенного предыдущей настройкой. Этот настройкой определяет, как быстро больше токены приближаются к этому пределу

Предыдущая настройка определяет самый большой возможный дождь, то есть верхний предел для размера любого дождя. Половина этого предела называется дождем среднего размера.

Эта настройка определяет, сколько токенов составляют средний дождь. Меньшее количество токенов вызывает меньший дождь, как и ожидает посетитель. Большее количество токенов вызывает немного больший дождь, приближаясь к пределу.

Más fichas provocan más lluvia, hasta un límite definido por el ajuste anterior. Este ajuste determina la rapidez con la que más fichas se acercan a ese límite.

El ajuste anterior definía la mayor lluvia posible, es decir, el límite superior del tamaño de cualquier lluvia. La mitad del límite se denomina lluvia de tamaño medio.

Este ajuste define cuántas fichas se necesitan para obtener una lluvia de tamaño medio. Menos fichas provocan menos lluvia, tal y como esperaría un visitante. Más fichas provocan algo más de lluvia, acercándose al límite.

Mehr Tokens ergeben mehr Regen, bis zum Limit aus der vorigen Einstellung. Diese Einstellung bestimmt, wie schnell mehr Tokens diesem Limit näher kommen.

Die vorige Einstellung bestimmt den größtmöglichen Regen, also das obere Limit für die Größe jedes möglichen Regens. Die Hälfte dieses Limits ist ein mittlerer Regen.

Diese Einstellung bestimmt, wie viele Tokens einen mittleren Regen ergeben. Weniger Tokens ergeben weniger Regen, genau wie die Besucher es erwarten. Mehr Tokens ergeben etwas mehr Regen, immer näher ans Limit.

Configuration Конфигурация Configuración Konfiguration

This app does not use fixed brackets, like: 50-99 tokens cause light rain, 100+ tokens cause heavy rain.

Instead, more tokens always cause more rain.

You configure the largest possible rain. More and more tokens get closer and closer to that largest rain.

В этом приложении не используются жесткие уровни, такие как: 50-99 токенов для легкого дождя, 100+ токенов для сильного дождя.

Больше токенов - больше дождя.

Вы определяете, насколько большим может быть дождь. Большее количество жетонов приводит к большему дождю, ближе к максимальному.

Esta aplicación no utiliza niveles rígidos, como: 50-99 fichas para lluvia ligera, 100+ fichas para lluvia fuerte.

Más fichas hacen más lluvia.

Se configura la mayor lluvia posible. Más y más fichas resultan en más y más lluvia, cada vez más cerca del máximo.

Diese App benutzt keine starren Stufen, wie: 50-99 Tokens für leichten Regen, 100+ Tokens für starken Regen.

Mehr Tokens ergeben immer mehr Regen.

Du konfigurierst den größten möglichen Regen. Mehr und mehr Tokens ergeben immer mehr Regen, immer näher am Maximum.
